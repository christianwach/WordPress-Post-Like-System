== Description ==

A simple and efficient post like system for WordPress. [View the demo.](http://jonmasterson.com/post-like-demo/)

### Plugin Development

This plugin is still in development. For feature requests and bug reports (or if you're a plugin author and want to contribute) please visit the plugin's [GitHub repository](https://github.com/JonMasterson/WordPress-Post-Like-System).



== Installation ==

1. Extract the plugin archive
1. Upload plugin files to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
